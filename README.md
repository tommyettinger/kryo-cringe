# kryo-cringe
Kryo 5.x support for classes from [cringe](https://github.com/tommyettinger/cringe)

## Get it

Use JitPack.io as your repository (which gdx-liftoff projects already can do) and depend on:
```
implementation "com.github.tommyettinger:kryo-cringe:0.3.1.2"
```

This depends on Kryo 5.6.2 and libGDX 1.14.2 .

## Older Versions

See [kryo-more](https://github.com/tommyettinger/kryo-more) for older releases.
